# pxlot - 機能一覧

Web ベースのピクセルアートエディタ（Rust + WebAssembly / Leptos）

---

## 描画ツール

| ツール | ショートカット | 説明 |
|--------|---------------|------|
| Pencil | B | 1ピクセル描画。ドラッグで連続線（Bresenham） |
| Eraser | E | 消しゴム。透明色で描画 |
| Fill | G | バケツ塗りつぶし（BFS）。上限 262,144px |
| Eyedropper | I | キャンバスから色を取得 |
| Line | L | 直線（Bresenham）。ドラッグでプレビュー表示 |
| Rectangle | R | 矩形アウトライン。ドラッグでプレビュー表示 |
| Ellipse | O | 楕円アウトライン（中点楕円アルゴリズム） |
| Filled Rectangle | Shift+R | 塗りつぶし矩形。ドラッグでプレビュー表示 |
| Filled Ellipse | Shift+O | 塗りつぶし楕円。ドラッグでプレビュー表示 |
| Rect Select | M | 矩形選択。破線で選択範囲表示 |

- ロック中・非表示レイヤーへの描画はブロックされ、ステータスバーに理由を表示
- **右クリック**: どのツール選択中でも消しゴムとして動作

## キャンバス操作

- **新規キャンバス**: ダイアログでサイズ指定（プリセット: 16/32/64/128）
- **キャンバスリサイズ**: 既存の内容を保持したままサイズ変更
- **ズーム**: マウスホイールで 1x〜64x（カーソル位置を中心にズーム）
- **パン**: マウスホイール押し込み（中ボタン）でドラッグ移動
- **グリッド表示**: チェックボックスで切替。ズーム 4x 以上で表示
- **グリッドサイズ**: 1px / 2px / 4px / 8px / 16px から選択可能
- **チェッカーボード**: 透明部分に市松模様を表示
- **ミラー描画**: 有効時、垂直軸で左右対称に描画（赤い軸線を表示）
- **タイルプレビュー**: タイリング表示の切替（チェックボックス）
- **初期サイズ**: 32x32 ピクセル
- **最小サイズ**: 1x1 ピクセル
- **最大サイズ**: 256x256 ピクセル

## レイヤー

| 機能 | 説明 |
|------|------|
| レイヤー追加 | `+` ボタン。自動で名前付与（Layer N） |
| レイヤー削除 | `x` ボタン。最後の1枚は削除不可 |
| 表示/非表示 | V/- ボタンで切替 |
| 不透明度 | 0〜255 のスライダー |
| レイヤー選択 | クリックでアクティブ化 |
| レイヤー複製 | 「〇〇 copy」として複製 |
| レイヤー並び替え | ▲/▼ ボタンで上下移動 |

- **メモリ上限**: 全レイヤー合計 256 MB
- **ブレンドモード**: Normal

---

## カラーピッカー

- **Hex入力**: `#RRGGBB` 形式。不正値で赤枠表示
- **RGBスライダー**: R/G/B 各チャンネルを 0〜255 で個別調整
- **前景色/背景色**: X キーで入れ替え
- **デフォルトパレット**: 32色。選択中の色にインジケータ表示
- **カラー履歴**: 直近32色を自動記録（透明色は除外、重複排除）

---

## 選択と編集

- **矩形選択**: M キーで選択モード。ドラッグで範囲指定
- **コピー**: Ctrl+C で選択範囲をクリップボードにコピー
- **カット**: Ctrl+X で選択範囲をコピーし透明に
- **ペースト**: Ctrl+V でクリップボードの内容を選択位置（または原点）に貼り付け

| 機能 | 説明 |
|------|------|
| フレーム追加 | `+` ボタン（空白フレーム） |
| フレーム複製 | `D` ボタン |

- オニオンスキン: 前フレームは青色、次フレームは赤色で半透明表示
- フレーム切替時にUndoヒストリーはリセットされる

## エクスポート

### PNG
- **スケール**: 4x / 8x / 16x（デフォルト 8x）
- **形式**: RGBA 8bit
- **スケーリング**: Nearest-neighbor（ピクセルアートに最適）
- **ファイル名**: カスタム指定可能（デフォルト: `pxlot.png`）

### GIF
- **アニメーション**: 全フレームをエクスポート
- **ループ**: 無限リピート
- **パレット**: 最大256色に自動量子化
- **透明**: インデックス0を透明として予約
- **ファイル名**: カスタム指定可能（デフォルト: `pxlot.gif`）

### スプライトシート
- **形式**: 全フレームを横一列に並べた PNG
- **スケール**: 1x（等倍）
- **ファイル名**: `{名前}_sheet.png`

---

## インポート

### PNG インポート
- **対応形式**: RGB, RGBA, Grayscale, Grayscale+Alpha
- **サイズ上限**: 256x256 ピクセル（超過時は自動ダウンスケール）
- **ダウンスケール**: Nearest-neighbor、アスペクト比維持
- **結果**: 新しいキャンバスとして読み込み（現在のプロジェクトを置換）

---

## AI アシスタント

### Pixelate（ピクセル化）
- **サイズ**: 16 / 32 / 48 / 64 / 128 ピクセル
- **色数**: 4 / 8 / 16 / 32 / 64
- **ディザリング**: None / Floyd-Steinberg

### Palette（パレット抽出）
- **色数**: 4 / 8 / 16 / 32
- **アルゴリズム**: メディアンカット色量子化
- **適用**: 抽出したパレットでキャンバスの色を削減可能

---

## データ保存

### オートセーブ
- **保存先**: LocalStorage（`pxlot_autosave`）
- **間隔**: 最小3秒ごと
- **内容**: キャンバスデータ（JSON）
- **起動時**: 保存データがあれば自動復元

### プロジェクト保存
- **保存先**: IndexedDB（`pxlot_projects`）
- **保存名**: "default"
- **内容**: タイムライン全体（全フレーム + メタデータ）
- **サイズ上限**: 50 MB（40 MB で警告）

---

## Undo/Redo

- **最大コマンド数**: 500
- **メモリ上限**: 128 MB
- **粒度**: ユーザー操作1回 = 1コマンド
- **ピクセル単位**: 変更前後の色を記録
- **制限超過時**: 最も古いコマンドから自動削除

---

## 国際化

- **対応言語**: English / 日本語
- **検出**: ブラウザの `navigator.language` を使用
- **切替**: メニューバーの EN/JP ボタン
